Image Analysis Software Knowledge Base

FIJI / IMAGEJ
Keywords: Fiji, ImageJ, NIH Image, macro, plugin, BioFormats, Java
Fiji Is Just ImageJ - the most widely used open-source image analysis platform. Supports macros, plugins, and the BioFormats library for reading proprietary formats. Key functions include measurement, segmentation, colocalization, and 3D visualization.

CELLPOSE
Keywords: Cellpose, deep learning, segmentation, cell detection, nuclei, GPU, Omnipose
Cellpose is a deep learning-based cell segmentation tool using a generalist model. Works on diverse cell types without retraining. Omnipose variant handles elongated cells.

ILASTIK
Keywords: ilastik, machine learning, pixel classification, object classification, random forest, interactive
ilastik provides interactive machine learning for image classification and segmentation. Uses random forest classifiers trained through user annotation.

IMARIS
Keywords: Imaris, 3D visualization, surface rendering, tracking, spots, Bitplane
Imaris is commercial software for 3D/4D visualization, surface rendering, object tracking, and quantitative analysis of microscopy data.

ARIVIS
Keywords: arivis, big data, Vision4D, Pro, InViewR, large volume, cloud
arivis handles extremely large image data with tiled pyramidal formats. Includes AI segmentation and VR visualization capabilities.

NAPARI
Keywords: napari, Python, viewer, multidimensional, plugin ecosystem, interactive
napari is a fast, interactive, multi-dimensional image viewer for Python. Growing plugin ecosystem for analysis and machine learning integration.

QUPATH
Keywords: QuPath, pathology, whole slide, tissue analysis, biomarker, cell detection
QuPath is open-source software designed for digital pathology and whole slide image analysis. Includes machine learning classifiers and cell detection.

ZEN
Keywords: ZEN, Zeiss, acquisition, processing, tiles, stitching, deconvolution
ZEN is Zeiss's microscope control and image processing software. Handles acquisition, tiling, stitching, and basic analysis.

ELEMENTS
Keywords: NIS-Elements, Nikon, acquisition, analysis, modules, JOBS
NIS-Elements is Nikon's platform for microscope control and image analysis. Modular design with specialized analysis packages.

METAMORPH
Keywords: MetaMorph, Molecular Devices, automated imaging, journals, multi-dimensional
MetaMorph is established software for automated microscopy and image analysis with journal-based automation.

HUYGENS
Keywords: Huygens, SVI, deconvolution, PSF, restoration, GPU
Huygens provides high-quality deconvolution using measured or theoretical PSFs. GPU acceleration for fast processing.

AIVIA
Keywords: Aivia, deep learning, segmentation, tracking, 3D, automated analysis
Aivia uses deep learning for automated image analysis including cell segmentation, tracking, and morphological quantification.

PYTHON ECOSYSTEM
Keywords: Python, scikit-image, scipy, numpy, opencv, tensorflow, pytorch
Python scientific stack for custom image analysis. scikit-image for algorithms, scipy for processing, deep learning via tensorflow/pytorch.

MATLAB
Keywords: MATLAB, Image Processing Toolbox, MathWorks, scripting
MATLAB provides image processing capabilities through toolboxes. Widely used for custom algorithm development in research.

CELLPROFILER
Keywords: CellProfiler, high-throughput, pipeline, batch processing, measurement
CellProfiler builds modular pipelines for high-throughput image analysis. Extracts quantitative measurements from cell images.

STARDIST
Keywords: StarDist, nuclei segmentation, star-convex, deep learning, U-Net
StarDist uses star-convex polygons for robust nuclei segmentation. Deep learning model pretrained on diverse datasets.

TRACKMATE
Keywords: TrackMate, Fiji, particle tracking, cell tracking, spots, links
TrackMate is a Fiji plugin for single particle and cell tracking. Provides multiple detection and linking algorithms.
